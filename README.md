# Random Password Generator

**Автор:** Виктория Кондратович

## Описание

Приложение с графическим интерфейсом для генерации случайных паролей. 
Позволяет настраивать длину пароля, выбирать типы символов (цифры, буквы, спецсимволы), 
сохраняет историю сгенерированных паролей в JSON-файл и загружает её при запуске.

### Возможности

- Ползунок длины пароля от 4 до 50 символов
- Чекбоксы для выбора типов символов
- Генерация случайного пароля
- Копирование пароля в буфер обмена
- История последних 50 паролей с датой и временем
- Сохранение и загрузка истории в JSON
- Очистка истории
